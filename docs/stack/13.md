# Next Greater Element

Given an array `arr[]` of integers, determine the Next Greater Element (NGE) for every element in the array, maintaining the order of appearance. The Next Greater Element for an element $x$ is defined as the first element to the right of $x$ in the array that is strictly greater than $x$.

If no such element exists for an element, its Next Greater Element is $-1$.

## Example

```
 input = [1, 3, 2,  4]
output = [3, 4, 4, -1]

 input = [6,  8, 0, 1,  3]
output = [8, -1, 1, 3, -1]
```

## Solution

??? "Expand"

    Create a stack and process the elements one-by-one from right to left. For each element $e$ at index $i$:

    - remove all elements $x$ from the stack if $x \le e$.
    - top of the stack will have the NGE of $e$.
    - add $e$ to the stack.

    ```kotlin
    fun nextGreater(input: IntArray): IntArray {
      val stack = ArrayDeque<Int>()
      for ((i, n) in input.withIndex().reversed()) {
        while (stack.isNotEmpty() && stack.last() <= n) {
          stack.removeLast()
        }
        input[i] = if (stack.isEmpty()) -1 else stack.last()
        stack.add(n)
      }
      return input
    }
    ```

    package org.example

    import org.junit.jupiter.api.Assertions.assertArrayEquals

    class NextGreaterElementTest {

        val input = intArrayOf(1, 3, 2, 4)

        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test example 2`() {
        val input = intArrayOf(6, 8, 0, 1, 3)
        val expected = intArrayOf(8, -1, 1, 3, -1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test strictly increasing array`() {
        // Every element should find its immediate neighbor as the NGE
        val input = intArrayOf(10, 20, 30, 40)
        val expected = intArrayOf(20, 30, 40, -1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test strictly decreasing array`() {
        // No element has a greater element to its right
        val input = intArrayOf(40, 30, 20, 10)
        val expected = intArrayOf(-1, -1, -1, -1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test array with identical elements`() {
        // Since we look for "strictly greater", duplicates don't count
        val input = intArrayOf(5, 5, 5, 5)
        val expected = intArrayOf(-1, -1, -1, -1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test array with zig-zag pattern`() {
        val input = intArrayOf(10, 5, 11, 6, 12)
        val expected = intArrayOf(11, 11, 12, 12, -1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test single element array`() {
        val input = intArrayOf(100)
        val expected = intArrayOf(-1)
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test empty array`() {
        val input = intArrayOf()
        val expected = intArrayOf()
        assertArrayEquals(expected, nextGreater(input))
      }

      @Test
      fun `test array where last element is largest`() {
        val input = intArrayOf(1, 2, 3, 10)
        val expected = intArrayOf(2, 3, 10, -1)
        assertArrayEquals(expected, nextGreater(input))
      }
    }
