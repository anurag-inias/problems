# Valid Parentheses in an Expression

Given a string s containing three types of brackets `{}`, `()` and `[]`. Determine whether the Expression are balanced or not.

## Solution

??? "First approach"

    $O(n)$ time and $O(n)$ space.

    We process the input character by character. If we encounter any opening bracket, it goes straight to the stack.

    If it's a closing bracket, we check if a corresponding opening bracket is available at the top of the stack.

    ```kotlin
    fun isBalanced(s: String): Boolean {
      val stack = ArrayDeque<Char>()

      for (c in s) {
        if (c.isOpening) {
          stack.add(c)
          continue
        }
        if (stack.isEmpty()) return false
        if (stack.removeLast() != c.opening) return false
      }

      return stack.isEmpty()
    }

    private val Char.isOpening: Boolean
      get() = this in listOf('(', '{', '[')

    private val Char.opening: Char
      get() = when(this) {
        ')' -> '('
        '}' -> '{'
        ']' -> '['
        else -> '\n'
      }
    ```

??? "Second approach"

    This will not really work if the language does not support mutable string. In kotlin/Java, we can mimic this by treating the string as a character array.

    We treat out input string itself as a stack. A pointer `top` tracks the top of the stack where the opening brackets are stored.

    ```kotlin
    fun isBalanced(s: String): Boolean {
      return isBalancedArray(s.toCharArray())
    }

    private fun isBalancedArray(s: CharArray): Boolean {
      var top = 0 // (1)
      for (c in s) {
        if (c.isOpening) {
          s[top++] = c // (2)
          continue
        }
        if (top == 0) return false // (3)
        if (s[--top] != c.opening) return false // (4)
      }
      return top == 0 // (5)
    }

    private val Char.isOpening: Boolean
      get() = this in listOf('(', '{', '[')

    private val Char.opening: Char
      get() = when(this) {
        ')' -> '('
        '}' -> '{'
        ']' -> '['
        else -> '\n'
      }
    ```

    1. same as `val stack = ArrayDeque<Char>()`
    2. same as `stack.add(c)`
    3. same as checking `stack.isEmpty()`
    4. same as checking `stack.removeLast() != c.opening`
    5. same as checking `stack.isEmpty()`
    

## Unit tests

??? "Expand"

    ```kotlin
    package org.example

    import org.junit.jupiter.api.Assertions.assertFalse
    import org.junit.jupiter.api.Assertions.assertTrue
    import org.junit.jupiter.api.Test

    class ParenthesesTest {

      @Test
      fun `empty string should be balanced`() {
        assertTrue(isBalanced(""))
      }

      @Test
      fun `simple matching parentheses should be balanced`() {
        assertTrue(isBalanced("()"))
      }

      @Test
      fun `multiple matching types should be balanced`() {
        assertTrue(isBalanced("()[]{}"))
      }

      @Test
      fun `nested matching brackets should be balanced`() {
        assertTrue(isBalanced("[{()}]"))
      }

      @Test
      fun `complex valid expression should be balanced`() {
        assertTrue(isBalanced("((()))[{}]{}"))
      }

      @Test
      fun `single opening bracket should not be balanced`() {
        assertFalse(isBalanced("("))
      }

      @Test
      fun `single closing bracket should not be balanced`() {
        assertFalse(isBalanced("]"))
      }

      @Test
      fun `mismatched closing bracket should not be balanced`() {
        assertFalse(isBalanced("(]"))
      }

      @Test
      fun `interleaved brackets should not be balanced`() {
        // Even though types match, order is wrong
        assertFalse(isBalanced("[(])"))
      }

      @Test
      fun `closing bracket with empty stack should not be balanced`() {
        assertFalse(isBalanced("())"))
      }

      @Test
      fun `extra opening brackets at the end should not be balanced`() {
        assertFalse(isBalanced("{([])}("))
      }

      @Test
      fun `reversed brackets should not be balanced`() {
        assertFalse(isBalanced(")("))
      }
    }
    ```
