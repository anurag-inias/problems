# Stock Span Problem

Given an array `arr[]` of daily stock prices, the stock span for the $i^{th}$ day is the count of consecutive days up to and including day $i$, such that each of those days had a stock price less than or equal to the price on day $i$.

## Example

```
 input = [100, 80, 60, 120]
output = [1, 1, 1, 4]

 input = [10, 4, 5, 90, 120, 80]
output = [1, 1, 2, 4, 5, 1]
```

## Solution

??? "Expand"

    ```kotlin
    fun calculateSpan(input: IntArray): IntArray {
      val stack = ArrayDeque<Int>()
      val output = IntArray(input.size)

      for ((i, v) in input.withIndex()) {
        while (stack.isNotEmpty() && input[stack.last()] <= v) {
          stack.removeLast()
        }
        output[i] = if (stack.isEmpty()) {
          i + 1
        } else {
          i - stack.last()
        }
        stack.add(i)
      }
      return output
    }
    ```

    There are just two scenarios to understand:

    - stack gets emptied altogether. In which case the number of days before and including $i$ are just $i+1$.
    - stack is not actually empty, and stack top is $x$. That is $v_x > v_i$. So the number of days we are interested in are $|(x, \dots, i]| = i - x$.
