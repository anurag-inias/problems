# Reversing the first K of a Queue

Given an integer k and a queue q of integers, reverse the order of the first k elements of the queue, leaving the other elements in the same relative order.

## Example

```
 queue = [1, 2, 3, 4, 5]
     k = 3

result = [3, 2, 1, 4, 5]
```

## Solution

??? "Expand"

    Let's work through a concrete example of a queue:

    ```
    front = [1, 2, 3, 4, 5] = back
    ```

    in a queue, elements are enqueued at the `back` and dequeued at the front.

    ----

    For our solution, we first dequeue first $k$ elements and save them on a stack:

    ```
    queue = [4, 5]
    stack = [3, 2, 1] 
    ```

    next we pop and enqueue these elements back on the queue:

    ```
    queue = [4, 5, 3, 2, 1]
    stack = []
    ```

    and finally, the elements that we left on the queue, are rotated to the back of the queue using `dequeue-enqueue`.

    ```
    queue = [3, 2, 1, 4, 5]
    ```

    ----

    ```kotlin
    fun <T> reverseFirstK(queue: ArrayDeque<T>, k: Int) {
      // Remove first k elements and temporarily save them on a stack.
      val stack = ArrayDeque<T>()
      for (i in 1..k) {
        if (queue.isNotEmpty()) {
          stack.add(queue.removeFirst())
        }
      }

      // Count the remaining elements on the queue.
      // This will need to be rotated back to the queue later.
      val remaining = queue.size

      // Pop the stack and put back the elements in the queue,
      // which are now in reversed order.
      while (stack.isNotEmpty()) {
        queue.add(stack.removeLast())
      }

      // Take care of any elements that needs to be rotated to the back.
      // It will only execute if k < |queue|.
      for (i in 1..remaining) {
        queue.add(queue.removeFirst())
      }
    }
    ```

## Unit tests

??? "Expand"

    ```kotlin
    package org.example

    import kotlin.collections.ArrayDeque
    import org.junit.jupiter.api.Test
    import org.assertj.core.api.Assertions.assertThat

    class StackReverseTest {

      @Test
      fun `queue of 5 elements, front 2 are reversed`() {
        val queue = ArrayDeque<Int>()
        queue.addAll(listOf(1, 2, 3, 4, 5))

        reverseFirstK(queue, 2)

        assertThat(queue).containsExactly(2, 1, 3, 4, 5)
      }

      @Test
      fun `queue of 5 elements, front 4 are reversed`() {
        val queue = ArrayDeque<Int>()
        queue.addAll(listOf(1, 2, 3, 4, 5))

        reverseFirstK(queue, 4)

        assertThat(queue).containsExactly(4, 3, 2, 1, 5)
      }

      @Test
      fun `queue of 5 elements, all 5 are reversed`() {
        val queue = ArrayDeque<Int>()
        queue.addAll(listOf(1, 2, 3, 4, 5))

        reverseFirstK(queue, 5)

        assertThat(queue).containsExactly(5, 4, 3, 2, 1)
      }

      @Test
      fun `queue of 5 elements, asked to reverse more than 5 elements`() {
        val queue = ArrayDeque<Int>()
        queue.addAll(listOf(1, 2, 3, 4, 5))

        reverseFirstK(queue, 6)

        assertThat(queue).containsExactly(5, 4, 3, 2, 1)
      }

      @Test
      fun `queue of 5 elements, front 1 are reversed`() {
        val queue = ArrayDeque<Int>()
        queue.addAll(listOf(1, 2, 3, 4, 5))

        reverseFirstK(queue, 1)

        assertThat(queue).containsExactly(1, 2, 3, 4, 5)
      }

    }
    ```
